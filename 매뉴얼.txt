1. 파일 형식

2. 메타데이터
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  파일 맨 위에 작성. {키: 값} 형식.

  {title: 곡 제목}       ← 필수
  {artist: 아티스트}     ← 필수
  {key: G}               ← 원곡 키 (A~G, b/#, m 조합)
  {capo: 0}              ← 카포 (생략 가능)
  {bpm: 80}              ← 빠르기 (생략 가능)

3. 섹션 헤더
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  [Verse]       [Verse 1]     [Verse 2]
  [Chorus]      [Pre-Chorus]
  [Bridge]      [Interlude]   [Intro]   [Outro]

  ※ 대괄호 안에 코드가 아닌 텍스트면 자동으로 섹션 헤더 인식

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
4. 코드 표기 규칙
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ■ 기본: 코드를 대괄호로 감싸고 바로 뒤에 가사

    [G]너에게난   [Am7]사랑해   [F#m]기억

  ■ 한 마디에 코드 두 개 (점으로 구분):

    [G.D.]너에게 난        → G와 D가 한 마디 안에
    [Em.Bm]녘 노을처럼    → 끝 점 생략해도 됨
    [C.G.]우 후회 없이

  ■ 대소문자 동일, 끝 점 유무 동일:
    [G] = [g] = [G.] = [G..]    모두 같음
    [G.D.] = [G.D]              모두 같음

  ■ 마디 구분: / (슬래시)
  ■ 한 줄 마지막에 // 쓰면 → 4마디 채우고 나머지는 쉼마디(회색)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
5. 예시 라인
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  4마디 한 줄 (마지막 / 생략 또는 // 사용):
    [G]가사 / [D]가사 / [Em]가사 / [C]가사 /

  2마디 + 쉼마디 2칸 (//) :
    [G.D.]가사 / [Em.C.]가사 //

  가사 없는 연주 구간:
    [G] / [D] / [Em] / [C] /
    [G.D.] / [Em.Bm.] / [C.G.] / [Am.D.] /

  인트로 짧은 라인 + 쉼마디:
    [C.D.] //       ← 1마디만 치고 3마디 쉼

## 출력 형식

```
export default `{title: 곡 제목}
{artist: 아티스트}
{key: 원곡 키}
{capo: 0}
{bpm: BPM}

[섹션명]
[코드]가사 / [코드]가사 / [코드]가사 / [코드]가사 /
`;
```

## 규칙

1. 메타데이터: {title}, {artist}, {key}, {capo}, {bpm} 순서로 맨 위에 작성
2. 섹션 구분: [Verse], [Chorus], [Bridge], [Interlude], [Outro] 등 대괄호로 표시
3. 코드 표기:
   - 한 마디에 코드 하나: [G]가사
   - 한 마디에 코드 둘: [G.D.]가사 (점으로 구분, 끝에 점 붙임)
   - 마디 구분은 / 사용
4. 가사는 해당 코드 대괄호 바로 뒤에 붙여 씀
5. 한 줄 = 4마디 기준. 마지막에 // 쓰면 나머지 마디 쉼표 처리
6. 연주 구간(가사 없음): [G.D.] / [Em.Bm.] / 형태로 코드만 표기
7. 대소문자 상관없이 인식되나, 가능하면 표준 표기 사용 (C, Dm, Em, F, G, Am 등)
8. 원본 악보의 코드 진행을 그대로 유지. 가사는 해당 마디에 맞게 배치
